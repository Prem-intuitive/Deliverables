# Week 1 - Semantic Profile Page

## Objective

Build a simple semantic profile page using HTML5 and basic CSS.

## Technologies Used

- HTML5
- CSS3
- Git

## Project Structure

```
index.html
style.css
README.md
```

## Features

- Semantic HTML
- Accessible page structure
- Minimal CSS styling
- Responsive viewport

## Accessibility Checklist

- Semantic HTML elements
- Proper heading hierarchy
- Language attribute added
- Viewport meta tag
- Accessible section labels using aria-labelledby
- Readable text

## Git Commands Used

```bash
git init

git add .

git commit -m "Week 1 Semantic Profile Page"
```
