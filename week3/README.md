# Week 3 - Interactive JavaScript Quiz App

## Objective

Build an interactive quiz application using JavaScript and DOM APIs to practice JavaScript/TypeScript fundamentals.

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

```text
index.html
style.css
script.js
README.md
```

## Features

* Interactive multiple-choice quiz
* Real-time score tracking
* Quiz timer
* Dynamic DOM updates
* Next question navigation
* Quiz result summary with score and time
* Restart quiz functionality

## Concepts Covered

* Variables and data types
* Arrays and objects
* Functions
* Control flow
* DOM manipulation
* Event handling
* Timers (`setInterval`)
